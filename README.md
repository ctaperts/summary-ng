# Summary Angular

Node/angular application to summarize pdf documents

## Setup node environment secrets
copy `nodemon.json.example` to `nodemon.json`

## Packages required
python
```
pip install nltk pika
```
node/angular
```
npm i .
```

## Development server
start together
```
npm run dev
```

start individually
```
ng serve
python ./mq-python/server.py
npm run dev-node
```
